# Daily-Task-Tracker
Daily Task Tracker is a productivity-focused web application that helps users organize, manage, and monitor their daily tasks efficiently. Users can create and manage multiple tasks, mark them as completed, and track their progress through interactive visualizations.

Features
Add, edit, and delete multiple daily tasks
Track task completion status in real time
Visualize productivity with dynamic Pie Charts and Bar Graphs
Monitor daily progress and completion trends
Clean and user-friendly interface for effective task management

This project is designed to help users stay organized, improve productivity, and gain insights into their daily performance through data-driven analytics.
